# Como usar o MinerHunter-DOIZZP

## Passo rapido

1. Extraia o ZIP.
2. Coloque a pasta em `C:\MinerHunter-DOIZZP`.
3. Execute `Run-MinerHunter.bat` como administrador.
4. Escolha `[1] Diagnostico completo`.
5. Abra a pasta `Reports`.
6. Analise `Suspects_DATA.csv`.

## Quando desconfiar de miner

- CPU/GPU alta com PC parado.
- Consumo cai ao abrir Gerenciador de Tarefas.
- Ventoinhas aceleram sem motivo.
- Quedas fortes de FPS em jogos.
- Processos estranhos em AppData/Temp/ProgramData.

## Procedimento seguro

1. Rode o diagnostico.
2. Nao apague nada.
3. Confira caminho, assinatura e nome do processo.
4. Verifique inicializacao no Autoruns.
5. Desabilite inicializacao suspeita.
6. Reinicie.
7. Se confirmado, use quarentena.
8. Rode Defender Offline.
